## Why this release matters

`prtr` is not just a one-off prompt helper. It is becoming a repeatable AI work loop.

You can start with a multilingual request, route it to Claude, Codex, or Gemini, and keep the next step cheaper with `swap`, `take`, `again`, `learn`, and `inspect`.

## Launch path

1. Open the project site
2. Run `prtr setup`
3. Run `prtr doctor`
4. Send the first prompt with `prtr go`
5. Keep the loop moving with `swap`, `take`, `again`, `learn`, and `inspect`
