섹션 3: --deep 파이프라인 시각화

prtr take patch --deep 실행 시 5단계 파이프라인을 어떻게 보여줄까?

$ prtr take patch --deep
⠸ planner ✓ 완료
⠼ patcher ⠼ 실행 중...
critic 대기 중
tester 대기 중
reconciler 대기 중

A. 단계별 스피너 리스트

각 단계가 순서대로 체크됨. 가볍고 빠름. bubbletea 최소 활용.

$ prtr take patch --deep
2/5
planner → patcher → critic → tester → rec
patcher: 변경 범위 분석 중 (3s)

B. 프로그레스바 + 파이프 시각화

전체 진행률 바 + 현재 단계 하이라이트. 바이럴 GIF 찍기 좋음.

$ prtr take patch --deep
✓ planner
scope: auth.js 수정, 테스트 2개 타겟
⠼ patcher 실행 중...
diff 생성 중: login() 함수 재작성
○ critic

C. 단계별 라이브 출력

각 단계의 중간 결과를 실시간으로 스트리밍. 가장 투명하지만 출력이 길어짐.
